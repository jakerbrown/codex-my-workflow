# Proofread Report: roth blog post

- **Date:** 2026-04-08
- **Reviewer role:** proofreader
- **Status:** findings addressed

## Primary findings

1. The draft used language that was too certain for an excerpt-based,
   *New Yorker*-only corpus.
2. The Goodreads section mixed a pending verification note with a recommendation
   list, which made the personalization status confusing.
3. The common-wisdom section needed to read more clearly as a secondary context
   section rather than a broader empirical test.

## Fixes applied

- Replaced over-confident wording with corpus-bounded language such as “in this
  corpus” and “in this run.”
- Split the Goodreads material into:
  - `Pending Goodreads overlap`
  - `Provisional read-next shortlist`
- Tightened the common-wisdom section so its limits are explicit up front.

## Residual note

The draft is now much cleaner about what is empirical, what is provisional, and
what remains blocked.
