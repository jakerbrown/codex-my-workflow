# Adversarial Review Round 1: Spencer underdog betting

## Main attack points

### 1. The source window is old relative to the current date

The best reproducible cross-league public archive ends in 2021. That does not
invalidate the historical analysis, but it weakens any rhetoric that sounds
like a statement about today's market.

### 2. The regular-season boundary is proxy-based

The project is explicit about this, but the full-season benchmark remains less
clean than an official season-type flag would be.

### 3. The "big dog" refinement is vulnerable to p-hacking

The positive `big_dog_3w` result is exactly the sort of in-sample tail variant
that could arise from strategy screening.

### 4. The simulation is not calibrated enough to carry much evidentiary weight

It is a mechanism illustration, not a disciplined structural exercise.

## Required fixes before final sign-off

1. Keep the post anchored in historical evidence ending in 2021.
2. Keep saying "season proxy" when discussing the all-season benchmark.
3. Keep the big-dog variant clearly secondary.
4. Keep the simulation clearly interpretive.
