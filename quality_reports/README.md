# Quality Reports

This directory stores durable workflow artifacts.

Recommended subfolders:

- `plans/` — approved or in-progress task plans
- `session_logs/` — short running notes for state recovery
- `merges/` — merge-time summary reports
- review outputs written by skills and specialist agents

Typical files written here by the starter pack:

- `*_visual_audit.md`
- `*_pedagogy_report.md`
- `*_proofread_report.md`
- `*_slide_excellence.md`
- `*_qa_critic_roundN.md`
- `*_qa_fixer_roundN.md`
- `*_qa_final.md`

The point of this directory is that important context survives beyond a single
chat or session.
