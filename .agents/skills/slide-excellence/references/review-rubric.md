# slide-excellence review rubric

Use this as a rough synthesis rubric rather than a rigid grader.

## Status buckets

| Status | Typical profile |
|--------|------------------|
| EXCELLENT | 0-2 critical issues, only a handful of medium/minor issues, clearly teachable now |
| GOOD | no catastrophic problems, but several meaningful fixes still advisable |
| NEEDS WORK | multiple blocking or major issues across more than one dimension |
| POOR | deck needs structural revision or has strong parity / pedagogy / layout failures |

## Suggested score anchors

| Score band | Meaning |
|------------|---------|
| 95-100 | exceptional, near-publication / near-teaching quality |
| 90-94 | PR-ready or milestone-ready |
| 80-89 | solid production baseline |
| 60-79 | exploration / draft quality |
| <60 | not ready for production use |

## Severity guidance

- **Critical**: likely to break the deck, misteach material, or visibly fail in presentation
- **Major**: important quality defect that should be fixed before a milestone
- **Minor**: polish issue that does not block use on its own
