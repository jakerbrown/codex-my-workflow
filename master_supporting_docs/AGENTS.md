# Supporting documents guidance

This directory holds source material such as papers, slide decks, appendices,
and other reference PDFs.

## Safe handling rules

- Do not try to absorb a very large PDF all at once.
- Inspect size and page count first.
- Read selectively by section or page range.
- When summarizing, record exact page ranges or section names.
- Treat these files as evidence sources, not as material to duplicate wholesale
  into slides or the knowledge base.

## When using supporting documents

- Prefer extraction, comparison, and selective quotation over full-document
  transcription.
- If a PDF is especially large, chunk the task and keep the running summary on
  disk.
